# 🗣️ English Fluency App — मराठी लोकांसाठी

> A free, open-source 90-day English fluency mobile app built specifically for **Marathi speakers** — powered by psycholinguistics, spaced repetition, and real-world printing industry vocabulary.

**No ads. No cost. No login. Open to all. — मोफत, जाहिरात नाही, सर्वांसाठी.**

---

## ✨ What is this?

A science-based English learning companion app that transforms Marathi-speaking professionals into confident English communicators in 90 days.

Built on:
- 🧠 **Psycholinguistics** — how the brain actually learns language
- 🔁 **Spaced Repetition** — proven memory technique
- 🎯 **Active Recall** — vocabulary flashcards with Marathi translations
- 🏋️ **Daily Drill System** — 20-minute structured speaking practice
- 🖨️ **Printing Industry Vocabulary** — real-world professional English

---

| Feature | Description |

---

## 🗺️ The 3-Phase System

### Phase 1 — Foundation (Days 1–30) 🌱
**पाया — Activation & Habit Building**
- Break the speaking freeze
- Mirror Narration, 60-Second Blast, Sentence Expansion
- Bridge Phrases to replace silence
- 18+ Printing industry vocabulary words

### Phase 2 — Fluency Building (Days 31–60) 🚀
**प्रवाहितता — Speed, Structure & Professional Power**
- PREP Framework (Point → Reason → Example → Point)
- AI Conversation Simulation Drills
- Shadowing technique
- Email-to-Speech bridge

### Phase 3 — Advanced Communication (Days 61–90) 👑
**प्रगत संवाद — Command, Lead & Own Your Voice**
- Pressure Drills (5-second response training)
- Presentation Framework
- Tone-Shift Drill (Formal / Friendly / Firm)
- English Voice Journal

## 🚀 Getting Started

### Run Locally

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/english-fluency-marathi.git
cd english-fluency-marathi

# Install dependencies
npm install

# Start development server
npm start
```

### Requirements
- Node.js 16+
- React 18+
- Runs in any modern browser

### Deploy to Vercel (Free)
```bash
npm install -g vercel
vercel
```

### Deploy to Netlify (Free)
```bash
npm run build
# Drag the /build folder to netlify.com/drop
```
